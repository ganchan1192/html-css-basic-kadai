忍者CODE甲賀ロードマップ

  1. 甲賀のSTEP1でHTMLとCSSの基礎を学ぶ。ホームページを完成させる流れを知る。
  2. 甲賀のSTEP2で模写コーディングを行ないHTMLとCSSを更に磨き上げる。JavaScriptの基礎も学ぶ。
  3. 甲賀のSTEP3でjQueryを学ぶ。SASSを学ぶ。JavaScriptでアプリを作る。
  4. 甲賀のSTEP4でJavaScriptを極める。ボリュームのあるサイトを作る。模写をする。
  5. 甲賀のSTEP5でPHPの基礎を学ぶ。WordPressを徹底的に学ぶ。
